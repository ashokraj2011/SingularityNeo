# Enterprise Workflow Templates

This folder contains reusable templates for the built-in agents, SDLC workflow steps, governance gates, and release approvals used by the capability workspace.

- `agents/`: role-specific operating templates for the built-in agents.
- `steps/`: SDLC step templates with hand-off and approval guidance.
- `workflows/`: end-to-end workflow definitions that can be used as standard capability baselines.
